#### 开发环境
- 语言：Java 8
- IDE(JAVA)： IntelliJ IDEA
- 依赖管理：Maven
- 镜像管理：Docker
- 数据库：MySQL5.7
### 项目涉及技术栈 `[持续更新中]`
1. SpringBoot 2.7.0
2. SpringCloud 2021.0.3
3. Swagger-UI API接口文档
4. Logback 日志
5. Nacos 服务注册/配置中心
6. Redis 缓存数据库
7. RocketMQ 消息队列
8. SpringBootAdmin 系统监控
9. Mybatis 持久层管理
10. Feign 微服务Http远程调用
11. Sentinel 熔断限流
12. SpringCloudGateway 网关
12. Coding...
